HackUTD — T-Mobile Admin Dashboard

A comprehensive admin dashboard built for competitive intelligence analysis and real-time notifications.

Frontend: React + TypeScript + Vite
Backend: Express + TypeScript
AI Integration: Nemotron + Parallels Web AI + Google Gemini
Notifications: Email (Nodemailer) + Telegram Bot

Core Capabilities:

  • Real-time sentiment analysis via WebSocket streaming
  • Competitive intelligence decision engine
  • Multi-channel notification system (Email & Telegram)
  • Admin tooling with AI-powered analytics

The Problem

Challenges in Network Operations & Customer Support

1. Fragmented Intelligence

Network operators lack centralized competitive intelligence and sentiment analysis tools to make informed decisions quickly

2. Delayed Notifications

Critical tower status updates and SOS alerts require immediate multi-channel distribution to reach the right stakeholders

3. Manual Analysis

Analyzing customer sentiment and network data manually is time-consuming and prone to human error

4. Support Ticket Bottlenecks

Traditional support ticket systems create friction between customers and resolution teams

The Solution

Unified Admin Dashboard with AI-Powered Intelligence

Real-Time Analytics

  • WebSocket-based sentiment streaming
  • Historical sentiment tracking and analytics
  • AI decision engine for competitive analysis

Smart Notifications

  • Telegram Bot: Instant tower status alerts to operations teams
  • Email Service: SOS alerts and automated support ticket creation
  • Multi-channel test and status verification endpoints

AI Integration

  • Server-side AI proxies (avoiding CORS, securing API keys)
  • Parallel AI data extraction + Google Gemini analysis
  • Automated competitive intelligence insights

Developer Experience

  • Clean REST API architecture with comprehensive health checks
  • Environment-based configuration for secure deployment
  • Local development support with hot-reload
